# 🚗 ParkWatch

> 🚀 Built as a full-stack project to solve real-world parking challenges using modern web technologies.

## 📖 Overview

ParkWatch is a full-stack web application designed to simplify parking management.
It allows users to locate available parking slots, monitor occupancy, and optimize parking usage.

The system aims to reduce time spent searching for parking and improve urban traffic efficiency.

---

## ✨ Features

* 🔍 Real-time parking slot availability
* 📊 Smart monitoring of parking spaces
* 🧠 Intelligent system for efficient parking usage
* 🌐 Full-stack web application
* 📱 Responsive UI for better user experience

## ⚙️ Installation & Setup

### Clone the repository

git clone https://github.com/rohithpabba10/ParkWatch.git
cd ParkWatch

### Backend setup

cd backend
pip install -r requirements.txt
python server.py

### Frontend setup

cd frontend
npm install
npm start
